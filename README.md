# FullCount

FullCount는 야구 입문자부터 팬까지 모두를 위한 챗봇 기반 야구 통합 정보 안내 웹사이트입니다.

경기 일정과 결과, 선수 정보, 야구 규칙, 구장 위치 및 편의시설 정보를 한 사이트에서 제공하며, 사용자는 챗봇을 통해 자연어로 질문하고 내부 링크를 통해 원하는 정보로 바로 이동할 수 있습니다.

본 프로젝트는 2025 캡스톤 디자인 졸업작품으로 제작되었습니다.

## 프로젝트 개요
- 프로젝트명: FullCount  
- 프로젝트 유형: 졸업작품  
- 프로젝트 주제: 챗봇 기반 스포츠 정보 제공 웹서비스

## 개발 환경 및 사용 기술
- Frontend : HTML5, CSS3, JavaScript  
- Backend : Node.js, Express
- Database : MySQL
- Chatbot : Deppseek 오픈소스 활용하여 자연어 처리 기반 챗봇 연동
- AI서버 : Together AI
- 기타 : Git / GitHub, VS Code  

## 기획 배경  
야구에 관심은 있지만 경기 규칙, 선수 정보, 관람 정보가 흩어져 있어 입문자와 초보 관람객이 정보를 얻기 어렵다는 문제의식에서 출발하였습니다.  
여러 사이트를 이동하지 않고 한 곳에서 야구 정보를 확인할 수 있는 통합 플랫폼의 필요성을 느껴 본 프로젝트를 기획하였습니다.

## 문제 정의  
- 야구 관련 정보가 여러 사이트에 분산되어 있음  
- 초보자는 규칙과 경기 흐름을 이해하기 어려움  
- 경기장 방문 시 내부 시설 정보 확인이 불편함

## 제작 방향 및 목표
- 챗봇을 통한 빠른 정보 탐색 제공  
- 사용자 질문에 대한 즉각적인 응답 및 페이지 연결  
- 경기, 선수, 규칙, 위치 정보를 통합 관리  
- 관람객 중심의 정보 제공으로 이용 편의성 향상

## 주요 기능
### 챗봇  
- 자연어 질문을 통해 야구 관련 정보 제공  
- 경기 일정, 결과, 선수 정보, 규칙 관련 질의 응답  
- 대화 중 사이트 내부 관련 페이지로 이동 가능  

### 경기 정보  
- 날짜 선택을 통한 경기 일정 조회  
- 경기 결과 및 이닝별 점수표 제공  
- 경기 라인업 정보 제공  

### 선수단 정보  
- 감독 및 코치 정보  
- 타자 및 투수 정보 제공  
- 선수 상세 페이지를 통한 프로필, 포지션, 소개 확인  

### 야구 규칙  
- 공격과 수비 규칙 설명  
- 야구 입문자를 위한 기본 규칙 안내  

### 위치 안내  
- 구장 오시는 길 안내  
- 내부 지도 제공  
- 음식점 및 편의시설 위치를 지도와 리스트로 제공  

### 고객 지원  
- 공지사항  
- FAQ  

### 관리자 페이지  
- 공지사항 등록 및 관리  
- 경기 결과 및 라인업 입력 관리

## 사이트 구성
### 사용자 페이지  
- 메인 페이지(챗봇)  
- 선수단 정보  
- 경기 정보  
- 야구 규칙  
- 위치 안내  
- 고객 지원  

### 관리자 페이지  
- 공지 관리  
- 경기 결과 및 라인업 관리

## 차별화 요소
- 챗봇 중심의 정보 탐색 구조  
- 야구 정보와 관람 정보를 통합한 플랫폼  
- 경기 관람 전 필요한 위치 및 편의시설 정보 제공

## 사용자 이용 흐름
1. 사이트 접속 후 챗봇과 상호작용  
2. 자연어 질문을 통해 원하는 정보 탐색  
3. 경기 정보, 선수 정보, 위치 안내 등 상세 페이지 이동  
4. 경기 관람 전 필요한 정보 확인
